
## User usage 

* Start
    ```bash
    docker compose -f ./compose.yaml up -d

    # Operate With WebUI :http://127.0.0.1:8000/static/index.html
    # Open api docs : http://127.0.0.1:8000/docs
    ```

* Shutdown service
    ```bash
    docker compose down
    ```
